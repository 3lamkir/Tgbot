# 🤖 Telegram Shop Bot

Модульный бот-магазин на aiogram 3.

---

## 📁 Структура файлов

```
bot/
├── main.py              — запуск бота
├── config.py            — конфигурация (токен, админы, группа)
├── database.py          — хранилище данных (JSON)
├── requirements.txt     — зависимости
├── handlers/
│   ├── start.py         — /start и главное меню
│   ├── price.py         — прайс и категории
│   ├── cart.py          — корзина и оформление заказа
│   ├── orders.py        — чеки / история заказов
│   ├── admin.py         — админ-панель
│   └── support.py       — поддержка
├── utils/
│   ├── keyboards.py     — все кнопки
│   └── helpers.py       — вспомогательные функции
└── data/                — создаётся автоматически
    ├── config.json      — токен, ID админов, ID группы
    ├── users.json       — пользователи
    ├── orders.json      — заказы
    ├── price.json       — прайс (категории и товары)
    ├── topics.json      — топики супергруппы
    └── settings.json    — тексты (приветствие, прайс)
```

## 🚀 Установка в UserLAnd (Ubuntu)

```bash
# 1. Обновить пакеты
pkg update && pkg upgrade -y

# 2. Установить Python
pkg install python -y

# 3. Перейти в папку бота
cd ~/bot

# 4. Установить зависимости
pip install -r requirements.txt

# 5. Запустить
python main.py
```

## ⚙️ Первый запуск — настройка

При первом запуске появится консоль настройки:

```
>> /addapibot
Введи токен бота: 123456:ABC-DEF...

>> /adduser
Введи Telegram ID администратора: 123456789

>> /start
```

Получить свой Telegram ID можно у бота @userinfobot.

## 📋 Команды бота

### Для пользователей:
| Команда | Действие |
|---------|----------|
| `/start` | Главное меню |
| `/success` | Подтвердить выполнение заказа |
| `/stop` | Отменить текущий заказ |

### Для администраторов:
| Команда | Действие |
|---------|----------|
| `/admin` | Открыть админ-панель |
| `/bindgroup -100XXXXXXXXXX` | Привязать супергруппу |

## 💳 Способы оплаты

| Код | Валюта | Комиссия |
|-----|--------|----------|
| 1 | Рубли (₽) | 0% |
| 2 | Звёзды Тг (⭐) | +15% |
| 3 | Другая валюта 💳 | +10% |

## 🔗 Привязка супергруппы

1. Создай супергруппу с включёнными **Темами (форум)**
2. Добавь бота как администратора с правом управления темами
3. В боте выполни: `/bindgroup -100XXXXXXXXXX`
   (ID группы можно узнать переслав сообщение из группы боту @getmyid_bot)

## ✏️ Редактирование прайса

Через **AdminPanel → Редактор прайса** можно:
- Добавлять/удалять категории
- Добавлять/удалять/редактировать товары
- Менять: название, цену, минимум, способы оплаты, подсказку target

---

## 💾 Данные

Все данные сохраняются в папке `data/` в JSON-файлах.  
При перезапуске бота данные **не теряются**.

Если нужно сбросить только прайс — удали `data/price.json`.  
Если нужно сбросить всё — удали папку `data/`.

## ⚠️ Если что-то пошло не так

Замени только нужный файл из репозитория, например:
```bash
# Только cart.py
cp ~/новый_cart.py ~/bot/handlers/cart.py
python main.py
```
